
## DictSearch

A JS script that adds a custom searchbox for use on [wiki.anpang.lol](https://wiki.anpang.lol/) conlang dictionaries.

See [here](https://wiki.anpang.lol/misc/DictSearch) for more info and [here](https://wiki.anpang.lol/oly/Dictionary#Search) for an example usage on the Oluyek dictionary.
